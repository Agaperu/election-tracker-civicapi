# civicAPI Election Dashboard (React + Vite + JS)

## Requirements
- Node 18+ recommended

## Setup
1) Install deps
   npm install

2) Run dev (server + client)
   npm run dev

- Server runs: http://localhost:5179
- Client runs: http://localhost:5173

## What it does
- Search races by date range / query / country / province / district
- Select a race and show live results
- Auto-poll the race endpoint at a configurable interval
- Optional: show race history timestamps + fetch a snapshot
